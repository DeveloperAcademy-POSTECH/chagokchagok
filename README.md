# :iphone: 차곡차곡 (ChagokChagok)

드라이브를 하면서 나만의 드라이브 코스와 이색적인 장소를 '차곡차곡' 모아보는 App


## :pushpin: Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform
